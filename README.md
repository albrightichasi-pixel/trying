# Trying

A small practice folder with simple HTML games, a playful loan-summary page, and a Python weight converter.

## Files

- `index.html` - loan request demo page.
- `games.html` - game hub.
- `snake.html` - playable Snake game.
- `tictactoe.html` - playable Tic Tac Toe game.
- `app.py` - command-line weight converter.

Open the HTML files in a browser. Run the Python file from a terminal with `python app.py`.
